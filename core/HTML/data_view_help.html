数据查看器使用说明

1.历史导入/处理查看

这是以表格的形式呈现的所有导入\处理过的数据，包含一些基本信息。
  • type：一共就两种：Data 导入的数据 ProcessedData经过程序处理的数据
  • name：数据的名称，经过一定规则得到。导入数据取文件或文件夹名称
  • 序号：即导入数据的位次
  • 数据格式：导入时数据的格式
  • 处理类型：是用什么方法处理得到的，如果看不懂的话告诉我一下，我后面加对照表
  • 数据大小：其实是数据的尺寸，一般三维数据是时间×高度×宽度
  • 数据源：从哪里处理来的
  • 操作：目前有设为当前和显示选择两种 前者可以调换当前操作的数据（默认为最新导入和处理的数据）

2.所有数据详情查看

是更加详细的，以树的形式呈现的数据链。包含所有数据的几乎所有信息以及参数
提示：如果没能快速找到你的数据，可以点击展开所有。
  • 名称/key：数据的名称 / 或者数据下的参数Parameters 导入数据的参数 Other Results处理数据的参数
    如果名称看不懂的话告诉我一下，我后面加对照表
  • 类型：导入的类型或者处理的类型
  • 尺寸/大小：数据的尺寸（维度） / 以及数据的大小（MB格式），只有三维数据会计算大小
  • 数值范围：数值类型数据的最大值到最小值
  • 创建时/值：此数据创建或处理的时间 / 各类参数的值
  • 操作：目前有数据可视化可以使得二维数据快速显示在右侧自由灵活的数据结果窗口
