CWT(连续小波变换)分析

1. 功能概述

CWT(Continuous Wavelet Transform)是一种时频分析方法，通过不同尺度的小波函数分析信号的局部特征。相比STFT，CWT在时频分辨率上具有更好的灵活性。

2. 算法原理

CWT通过将信号与小波基函数进行卷积变换：

C(a, b) = 1/√a ∫-∞∞ x(t) ψ*((t-b)/a) dt

其中：

  • a：尺度参数（与频率相关）
  • b：平移参数（与时间相关）
  • ψ(t)：小波基函数

3. 参数说明

参数 类型 说明
wavelet str 小波基函数类型（如'morl', 'cmor'等）
scales array 尺度参数数组
sampling_period float 采样间隔时间
freq_range tuple 目标频率范围(min, max)

4. 小波基函数

  • Morlet小波：适用于频率分析
  • Mexican Hat小波：适用于信号奇点检测
  • Complex Morlet小波：提供相位信息
